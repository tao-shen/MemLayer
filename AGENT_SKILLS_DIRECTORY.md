## Installation Guide

### Using the Skills CLI (recommended)

```bash
# Install any skill from GitHub
npx skills add <owner/repo>

# List available skills in a repo without installing
npx skills add <owner/repo> --list

# Install a specific skill
npx skills add <owner/repo> --skill <skill-name>

# Install globally (available in all projects)
npx skills add <owner/repo> -g

# Target a specific agent
npx skills add <owner/repo> --agent cursor
npx skills add <owner/repo> --agent claude-code

# Search for skills
npx skills find <query>

# List installed skills
npx skills list

# Update all installed skills
npx skills update
```

### Manual Installation

Skills can be placed in the following directories depending on your agent:

| Agent | Project Path | Global Path |
|-------|-------------|-------------|
| Cursor | `.cursor/skills/` | `~/.cursor/skills/` |
| Claude Code | `.claude/skills/` | `~/.claude/skills/` |
| Codex | `.codex/skills/` | `~/.codex/skills/` |
| GitHub Copilot | `.github/skills/` | `~/.copilot/skills/` |
| Gemini CLI | `.gemini/skills/` | `~/.gemini/skills/` |
| Antigravity | `.agent/skills/` | `~/.gemini/antigravity/skills/` |
| OpenCode | `.opencode/skills/` | `~/.config/opencode/skills/` |
| Windsurf | `.windsurf/skills/` | `~/.codeium/windsurf/skills/` |
