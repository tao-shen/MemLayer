# 常见问题解答 (FAQ)

## 一般问题

### 什么是 Chat Interface？

Chat Interface 是一个智能对话界面，允许用户与 AI Agent 进行交互，并提供记忆可视化和区块链集成功能。

### 支持哪些浏览器？

推荐使用以下浏览器的最新版本:
- Google Chrome (推荐)
- Mozilla Firefox
- Microsoft Edge
- Safari 14+

### 是否支持移动设备？

是的，应用采用响应式设计，支持移动设备访问。但某些复杂的可视化功能在桌面端体验更好。

## 账户和会话

### 如何创建账户？

目前应用使用 Agent ID 进行身份识别，无需单独创建账户。

### 会话数据保存在哪里？

会话数据保存在服务器端，与您的 Agent ID 关联。

### 可以创建多少个会话？

没有硬性限制，但建议定期清理不需要的会话以保持界面整洁。

### 如何备份会话数据？

可以通过导出功能将数据导出为 JSON 或 CSV 格式进行备份。

## 聊天功能

### AI 回复的质量如何提高？

1. 使用清晰、具体的问题
2. 启用 RAG 模式利用历史记忆
3. 提供足够的上下文信息
4. 使用智能体 RAG 处理复杂问题

### 支持哪些消息格式？

- 纯文本
- Markdown 格式
- 代码块（带语法高亮）
- 链接（自动识别）

### 消息长度有限制吗？

单条消息建议不超过 4000 字符，过长的消息可能影响处理速度。

### 如何引用之前的对话？

AI 会自动记住会话上下文。启用 RAG 模式可以检索更早的对话内容。

## RAG 功能

### 什么是 RAG？

RAG (Retrieval-Augmented Generation) 是检索增强生成技术，通过检索相关记忆来提高 AI 回答质量。

### 标准 RAG 和智能体 RAG 有什么区别？

- **标准 RAG**: 使用向量相似度检索，适合简单问答
- **智能体 RAG**: 使用多步推理和策略选择，适合复杂任务

### RAG 会增加响应时间吗？

是的，RAG 需要额外的检索时间，通常增加 1-2 秒。但可以显著提高回答质量。

### 如何查看检索到的内容？

启用 RAG 后，AI 回复下方会显示"检索结果"卡片，点击可查看详细内容。

## 记忆可视化

### 什么是记忆类型？

- **STM (短期记忆)**: 临时对话内容
- **Episodic (情景记忆)**: 具体事件和经历
- **Semantic (语义记忆)**: 知识和概念
- **Reflection (反思记忆)**: 洞察和总结

### 时间线视图如何使用？

- 鼠标滚轮缩放
- 拖拽平移
- 点击节点查看详情
- 不同颜色代表不同记忆类型

### 知识图谱太复杂怎么办？

1. 使用过滤器减少节点数量
2. 调整重要性阈值
3. 搜索特定内容
4. 使用列表视图代替

### 如何导出可视化数据？

1. 点击"导出"按钮
2. 选择格式:
   - JSON: 完整数据
   - CSV: 表格数据
   - PNG/SVG: 图表图片
3. 下载文件

### 可视化性能优化建议

- 限制显示的记忆数量（使用过滤器）
- 关闭不需要的视图
- 使用列表视图处理大量数据
- 定期清理旧记忆

## 区块链功能

### 为什么要将记忆铸造为 NFT？

- 永久保存重要记忆
- 防篡改和可验证
- 可转移和交易
- 去中心化存储

### 支持哪些区块链？

目前支持 Solana 区块链（Devnet 和 Mainnet-Beta）。

### 铸造需要什么？

1. Solana 钱包（如 Phantom）
2. 足够的 SOL 支付交易费
3. 足够的 SOL 支付 Arweave 存储费

### 铸造费用是多少？

费用包括两部分:
- Solana 交易费: 约 0.000005 SOL
- Arweave 存储费: 根据数据大小，通常 0.001-0.01 SOL

### 铸造需要多长时间？

通常需要 30-60 秒，包括:
1. 加密数据 (5-10秒)
2. 上传到 Arweave (10-20秒)
3. 铸造 NFT (10-20秒)
4. 确认交易 (5-10秒)

### 铸造失败的常见原因

1. **余额不足**: 确保钱包有足够的 SOL
2. **网络拥堵**: 等待后重试
3. **用户拒绝**: 在钱包中确认交易
4. **RPC 错误**: 切换 RPC 端点

### 如何查看我的 NFT？

1. 在应用的"区块链"视图中查看
2. 在 Solana Explorer 中查看
3. 在钱包中查看（部分钱包支持）

### NFT 可以转移吗？

是的，可以转移给其他 Solana 地址。在资产详情中点击"转移"按钮。

### 如何设置访问控制？

在铸造时可以设置访问策略:
- 公开: 任何人可访问
- 私有: 仅所有者可访问
- 白名单: 指定地址可访问

### 数据存储在哪里？

- 元数据: Solana 区块链
- 完整内容: Arweave 永久存储
- 加密密钥: 链上管理

## 性能和优化

### 应用加载慢怎么办？

1. 检查网络连接
2. 清除浏览器缓存
3. 使用现代浏览器
4. 关闭不必要的浏览器扩展

### 如何提高可视化性能？

1. 减少显示的节点数量
2. 使用过滤器
3. 关闭不需要的动画
4. 使用列表视图代替图形视图

### 内存占用过高怎么办？

1. 关闭不需要的会话
2. 清除浏览器缓存
3. 重新加载页面
4. 减少可视化节点数量

## 安全和隐私

### 数据是否加密？

是的，敏感数据在传输和存储时都会加密。

### 谁可以访问我的数据？

只有您和授权的 Agent 可以访问您的数据。

### 如何删除我的数据？

可以删除会话和记忆。已铸造的 NFT 无法从区块链删除，但可以转移或销毁。

### 钱包安全建议

1. 使用硬件钱包（如 Ledger）
2. 不要分享私钥或助记词
3. 定期备份钱包
4. 谨慎确认交易

## 故障排除

### 无法连接到服务器

1. 检查网络连接
2. 确认服务器地址正确
3. 检查防火墙设置
4. 联系管理员

### WebSocket 连接失败

1. 检查浏览器是否支持 WebSocket
2. 检查代理设置
3. 尝试刷新页面
4. 检查服务器状态

### 钱包连接失败

1. 确保已安装钱包扩展
2. 刷新页面重试
3. 检查钱包是否解锁
4. 尝试其他钱包

### 页面显示异常

1. 清除浏览器缓存
2. 禁用浏览器扩展
3. 尝试无痕模式
4. 更新浏览器
