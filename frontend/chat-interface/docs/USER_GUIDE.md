## 快速开始

### 访问应用

1. 打开浏览器访问应用地址
2. 首次访问会看到欢迎界面
3. 点击"创建新会话"开始使用

### 创建第一个会话

1. 点击左侧边栏的"+"按钮
2. 选择 Agent 类型
3. 配置 RAG 模式（可选）
4. 点击"创建"按钮

## 功能概览

### 主要功能

- **智能对话**: 与 AI Agent 进行自然语言对话
- **RAG 增强**: 使用检索增强生成提高回答质量
- **记忆可视化**: 多维度查看和分析 Agent 记忆
- **区块链铸造**: 将重要记忆铸造为 NFT 永久保存

### 界面布局

- **左侧边栏**: 会话列表和管理
- **中间面板**: 聊天对话区域
- **右侧面板**: 记忆可视化和区块链资产

## 会话管理

### 创建会话

1. 点击左上角的"+"按钮
2. 填写会话配置:
   - **Agent 类型**: 选择适合的 Agent
   - **RAG 模式**: 选择检索模式
   - **记忆类型**: 选择要使用的记忆类型
3. 点击"创建会话"

### 切换会话

- 点击左侧边栏中的会话项即可切换
- 当前会话会高亮显示

### 管理会话

- **重命名**: 右键点击会话 → 选择"重命名"
- **删除**: 右键点击会话 → 选择"删除"
- **查看详情**: 点击会话右侧的信息图标

## 聊天功能

### 发送消息

1. 在底部输入框输入消息
2. 按 `Enter` 发送，`Shift + Enter` 换行
3. 等待 AI 回复

### RAG 模式

#### 关闭 RAG
- 标准对话模式，不使用记忆检索

#### 标准 RAG
- 使用向量检索相关记忆
- 适合一般问答场景

#### 智能体 RAG
- 使用高级检索策略
- 适合复杂推理场景

### 消息操作

- **复制**: 点击消息右上角的复制图标
- **重新生成**: 点击"重新生成"按钮
- **铸造 NFT**: 点击"铸造"按钮将消息保存到区块链

### 查看检索结果

当启用 RAG 模式时:
1. AI 回复下方会显示"检索结果"卡片
2. 点击展开查看详细内容
3. 相关性评分显示检索质量

## 记忆可视化

### 视图切换

点击右侧面板顶部的标签切换视图:

#### 时间线视图
- 按时间顺序显示记忆
- 支持缩放和滚动
- 高亮重要事件

#### 知识图谱视图
- 显示实体和关系网络
- 支持拖拽和缩放
- 点击节点查看详情

#### 列表视图
- 表格形式显示记忆
- 支持排序和过滤
- 快速浏览大量记忆

#### 统计视图
- 显示记忆统计信息
- 图表展示趋势
- 分析记忆分布

#### 区块链视图
- 显示已铸造的 NFT
- 查看资产详情
- 管理区块链资产

### 过滤和搜索

#### 使用过滤器

1. 点击"过滤器"按钮
2. 选择过滤条件:
   - **类型**: 选择记忆类型
   - **时间范围**: 选择日期范围
   - **重要性**: 调整重要性滑块
3. 点击"应用"

#### 搜索记忆

1. 在搜索框输入关键词
2. 系统会实时显示匹配结果
3. 搜索结果会高亮显示

### 查看记忆详情

1. 点击任意记忆节点或卡片
2. 右侧会显示详情面板
3. 查看完整信息和相关记忆

## 区块链集成

### 连接钱包

1. 点击右上角的"连接钱包"按钮
2. 选择钱包类型（Phantom、Solflare 等）
3. 在钱包中确认连接
4. 连接成功后显示钱包地址

### 铸造记忆 NFT

#### 从聊天消息铸造

1. 找到要铸造的消息
2. 点击消息右上角的"铸造"按钮
3. 确认铸造信息和费用
4. 在钱包中签名确认
5. 等待铸造完成

#### 从记忆列表铸造

1. 在可视化面板中选择记忆
2. 点击详情面板中的"铸造 NFT"按钮
3. 按照提示完成铸造

### 查看铸造进度

- 铸造过程会显示进度条
- 状态包括: 加密 → 上传 → 铸造 → 确认
- 完成后会显示交易签名

### 管理 NFT 资产

1. 切换到"区块链"视图
2. 查看所有已铸造的 NFT
3. 点击资产查看详情:
   - 元数据
   - Arweave URI
   - 交易记录
   - 访问控制策略

### 转移资产

1. 在资产详情中点击"转移"
2. 输入接收地址
3. 在钱包中确认交易
4. 等待转移完成

## 常见问题

### 聊天相关

**Q: 为什么 AI 回复很慢？**
A: 可能是网络延迟或服务器负载较高，请稍等片刻。

**Q: 如何删除消息？**
A: 目前不支持删除单条消息，但可以删除整个会话。

**Q: RAG 模式有什么区别？**
A: 标准 RAG 使用简单检索，智能体 RAG 使用高级策略，适合复杂场景。

### 可视化相关

**Q: 为什么图谱加载很慢？**
A: 大量节点会影响性能，建议使用过滤器减少显示数量。

**Q: 如何导出数据？**
A: 点击"导出"按钮，选择格式（JSON/CSV），下载文件。

**Q: 时间线视图如何缩放？**
A: 使用鼠标滚轮或触控板手势进行缩放。

### 区块链相关

**Q: 支持哪些钱包？**
A: 支持 Phantom、Solflare、Torus、Ledger 等主流 Solana 钱包。

**Q: 铸造需要多少费用？**
A: 费用包括 Solana 交易费和 Arweave 存储费，具体金额在铸造前会显示。

**Q: 铸造失败怎么办？**
A: 检查钱包余额是否充足，网络是否正常，可以重试铸造。

**Q: 如何查看我的 NFT？**
A: 在区块链视图中可以看到所有 NFT，也可以在 Solana 浏览器中查看。

### 性能相关

**Q: 应用运行卡顿怎么办？**
A: 
- 关闭不需要的浏览器标签
- 减少可视化节点数量
- 清除浏览器缓存
- 使用现代浏览器（Chrome、Firefox、Edge）

**Q: 移动端体验如何？**
A: 应用支持响应式设计，在移动端可以正常使用，但复杂可视化建议在桌面端使用。

## 键盘快捷键

- `Cmd/Ctrl + K`: 打开搜索

- `Cmd/Ctrl + /`: 显示快捷键帮助
- `Esc`: 关闭模态框
- `Enter`: 发送消息
- `Shift + Enter`: 换行
