# 记忆可视化 Demo

一个完整的记忆可视化演示应用，展示所有类型的记忆（STM、Episodic、Semantic、Reflection）。

## 功能特性

### 📊 四种可视化视图

1. **时间线视图** 📅
   - 按时间顺序展示所有记忆
   - 不同类型记忆用不同颜色和位置区分
   - 支持点击查看详情
   - 交互式 tooltip 显示摘要信息

2. **关系图谱** 🕸️
   - 力导向图展示记忆之间的关系
   - 三种关系类型：
     - 反思关系（紫色）：Reflection 记忆与源记忆的关联
     - 语义关系（橙色）：知识之间的概念关联
     - 时间关系（灰色）：事件的时间顺序
   - 支持拖拽节点
   - 支持缩放和平移
   - 节点大小表示重要性

3. **列表视图** 📋
   - 表格形式展示所有记忆
   - 支持多种排序方式：
     - 按时间排序
     - 按重要性排序
     - 按类型排序
   - 显示类型特定的详细信息
   - 快速浏览和搜索

4. **统计分析** 📊
   - 类型分布饼图
   - 重要性分布柱状图
   - 时间分布堆叠图（按周）
   - 详细统计数据
   - 统计摘要卡片

### 🎨 记忆类型

- **短期记忆 (STM)** - 蓝色 🔵
  - 当前会话的临时信息
  - 包含会话ID

- **情景记忆 (Episodic)** - 绿色 🟢
  - 具体事件和经历
  - 包含事件类型和访问次数

- **语义记忆 (Semantic)** - 橙色 🟠
  - 知识和概念
  - 包含来源、分类和验证状态

- **反思记忆 (Reflection)** - 紫色 🟣
  - 从其他记忆中提取的洞察
  - 包含洞察列表和源记忆引用

### 🔍 其他功能

- **类型过滤**：快速过滤显示特定类型的记忆
- **详情面板**：点击任何记忆查看完整详细信息
- **响应式设计**：适配不同屏幕尺寸
- **交互式可视化**：所有图表都支持交互操作

## 安装和运行

### 安装依赖

```bash
cd frontend/tacit knowledge-visualization-demo
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:3200` 启动。

### 构建生产版本

```bash
npm run build
```

## 技术栈

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **D3.js** - 数据可视化（时间线和图谱）
- **Recharts** - 统计图表
- **Tailwind CSS** - 样式框架
- **Vite** - 构建工具

## 模拟数据说明

Demo 包含了一个完整的学习场景数据集：

- **4 条短期记忆**：当前会话中关于机器学习的讨论
- **8 条情景记忆**：用户学习机器学习的完整历程
- **9 条语义记忆**：机器学习相关的知识和概念
- **5 条反思记忆**：从学习过程中提取的洞察和模式

所有记忆都有真实的时间戳、重要性评分和类型特定的元数据。

## 使用指南

### 1. 浏览时间线

- 在时间线视图中，记忆按时间从左到右排列
- 不同类型的记忆在不同的水平线上
- 鼠标悬停查看摘要，点击查看详情

### 2. 探索关系图谱

- 拖拽节点重新排列布局
- 使用鼠标滚轮缩放
- 拖拽背景平移视图
- 观察不同颜色的连线表示不同类型的关系

### 3. 使用列表视图

- 点击排序按钮改变排序方式
- 点击任意记忆卡片查看详情
- 快速浏览所有记忆的摘要信息

### 4. 查看统计分析

- 查看类型分布了解记忆构成
- 查看时间分布了解记忆增长趋势
- 查看重要性分布了解记忆质量
- 查看详细统计了解每种类型的具体数据

### 5. 过滤记忆

- 使用顶部的类型过滤器
- 快速切换显示特定类型的记忆
- 所有视图都会同步更新

## 扩展建议

如果要将此 Demo 连接到真实的后端 API：

1. 替换 `mockData.ts` 中的数据为 API 调用
2. 使用 React Query 或 SWR 进行数据获取
3. 添加 WebSocket 支持实时更新
4. 添加搜索和高级过滤功能
5. 添加数据导出功能
