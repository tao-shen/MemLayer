# 记忆可视化 Demo 快速启动指南

## 概述

我已经创建了一个完整的记忆可视化 Demo 应用，展示所有四种类型的记忆（STM、Episodic、Semantic、Reflection）。

## 🎯 Demo 特性

### 四种可视化视图

1. **时间线视图** 📅
   - D3.js 实现的交互式时间线
   - 按类型分层显示记忆
   - 节点大小表示重要性
   - 悬停显示详细信息

2. **关系图谱** 🕸️
   - 力导向图展示记忆关系
   - 支持拖拽、缩放、平移
   - 三种关系类型（反思、语义、时间）
   - 动态布局算法

3. **列表视图** 📋
   - 完整的记忆列表
   - 多种排序方式
   - 类型特定信息展示
   - 快速浏览

4. **统计分析** 📊
   - 类型分布饼图
   - 重要性分布柱状图
   - 时间分布堆叠图
   - 详细统计数据

### 记忆类型展示

- **短期记忆 (STM)** 🔵 - 4条
  - 当前会话信息
  - 会话ID标识

- **情景记忆 (Episodic)** 🟢 - 8条
  - 学习历程事件
  - 事件类型和访问次数

- **语义记忆 (Semantic)** 🟠 - 9条
  - 机器学习知识
  - 来源和分类信息

- **反思记忆 (Reflection)** 🟣 - 5条
  - 学习洞察
  - 源记忆关联

## 🚀 快速启动

### 1. 安装依赖

```bash
cd frontend/memory-visualization-demo
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问：`http://localhost:3200`

## 🎨 使用指南

### 切换视图

点击顶部的标签切换不同视图：
- 📅 时间线
- 🕸️ 关系图谱
- 📋 列表视图
- 📊 统计分析

### 过滤记忆

使用右上角的过滤按钮：
- 全部
- 短期记忆（蓝色）
- 情景记忆（绿色）
- 语义记忆（橙色）
- 反思记忆（紫色）

### 查看详情

点击任何记忆节点/卡片查看完整详细信息：
- 基本信息
- 类型特定字段
- 元数据
- 关联记忆

### 交互操作

**时间线视图：**
- 悬停查看摘要
- 点击查看详情

**图谱视图：**
- 拖拽节点调整位置
- 滚轮缩放
- 拖拽背景平移

**列表视图：**
- 点击排序按钮改变排序
- 点击卡片查看详情

## 🔧 技术实现

### 核心技术

- **React 18** + TypeScript
- **D3.js** - 时间线和图谱
- **Recharts** - 统计图表
- **Tailwind CSS** - 样式
- **Vite** - 构建工具

### 可视化算法

- **时间线**：线性时间映射 + 分层布局
- **图谱**：力导向算法（Force-directed）
- **统计**：数据聚合和分组

## 🐛 故障排除

### 端口冲突

如果 3200 端口被占用，修改 `vite.config.ts`：

```typescript
server: {
  port: 3201, // 改为其他端口
}
```

### 依赖安装失败

尝试清除缓存：

```bash
rm -rf node_modules package-lock.json
npm install
```
